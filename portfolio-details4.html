Project description

A dynamic and interactive social blogging platform designed to empower users to express themselves, connect with others, and explore a wide range of topics. With seamless integration of React, Tanstack for efficient data fetching and management, and a robust backend powered by Node.js and MySQL, React Blog offers a feature-rich experience for both bloggers and readers. Key Features: User Registration and Authentication: Users can register for an account and authenticate securely using email and password. Authentication features include login, logout, and password reset functionality. User Profile Management: Users can create and update their profiles, including profile pictures, bios, and other personal details. Profile information is stored securely and can be edited by the user at any time. Social Interactions: Users can follow other users to stay updated on their posts and activities. Users can view posts from users they follow on their personalized feed. Users can unfollow other users if they no longer wish to follow them. Blog Post Sharing: Users can create, edit, and delete their own blog posts. Posts can include text, images, and other multimedia content. Users can like, comment on, and share posts with their followers and friends. Content Discovery: Users can explore a variety of blog posts on different topics through a discover section. Posts are categorized and tagged for easy navigation and content discovery. Interactive Features: Users can like posts to show appreciation for the content. Users can comment on posts to share their thoughts and engage in discussions with other users. Marketplace Integration: React Blog includes a marketplace section where users can buy and sell goods and services. Users can browse products, view details, and make purchases securely through integrated payment gateways. Technology Stack: Frontend: React.js for building the user interface, Tanstack (React Query) for efficient data fetching and management, SCSS for styling. Backend: Node.js for server-side logic, Express.js for handling HTTP requests, MySQL for database management. Authentication: JWT (JSON Web Tokens) for secure authentication and session management. Other Tools: Axios for API requests, bcrypt for password hashing, Multer for handling file uploads.
